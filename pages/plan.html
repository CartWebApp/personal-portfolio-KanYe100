My Cart Web Application Journey

At first, I thought CART was for more intellectual individuals but I learned the acceptance to CART is random, anyone can get in. As a result, me and my friend applied and we wanted to get into the Web Application lab, thankfully we were both accepted and got what we wanted. I knew that the lab involved coding and I did not have any prior knowledge. Despite my lack on knowledge, I was eager to learn more.

We were first taught the basics of HTML and to me, it was very straightforward. However, I did not yet understand how this would result in a fully functional website. Then our teacher introduced CSS, which was the overall styling of HTML elements. Our teacher also assigned us many assignements that would further refine our CSS skills through projects or practices. I can firmly say that I have learned a signifcant amount via the projects and practices. CSS did help me understand the structure of a website more because it allowed my website to appear as a website but when I learned JavaScript, that was when everything clicked. Upon being introduced to JavaScript we were only using the console in the website, but when we actually began connecting the JavaScript to the HTML and CSS, I understood how a website is created.
